# Demo prompts

## 1. JD diagnosis / role calibration
“我们在帮一家消费科技公司招华东区销售 VP。CEO 希望这个人既能搭区域团队、又能自己拿大客户、还能推进渠道改革。这个岗位已经 open 了 4 个月。请你按 retained search 的视角，判断这个 mandate 到底卡在哪里，并给我一个可执行的画像调整建议。”

## 2. Sourcing strategy / executive search
“帮我给一家 Series C AI infra 公司做 VP Product 的 executive search strategy。创始人只想看既懂 enterprise product、又带过 20 人以上团队、还能和销售一起打大客户的人。请给我 target companies、adjacent backgrounds、off-limits 风险和前两周 search plan。”

## 3. Candidate assessment / shortlist gate
“我有个候选人之前在美团做过区域经营负责人，后来去一家新消费公司带全国渠道。现在客户想让他看 GM 级岗位。你按 shortlist gate 的标准评估一下：值不值得推进到 CEO 面？还要补问哪些关键问题？”

## 4. Search plan / multi-stakeholder mandate
“这个岗位不是单纯找人。董事会、CEO、CHRO 对画像都不完全一致，而且流程慢。请你帮我做一个 search plan：怎么校准 stakeholder、怎么控制 shortlist 节奏、什么时候应该建议客户调整 title / cash / scope。”

## 5. Intake-file demo
“请把这个 search mandate 按 executive-search intake schema 整理完整，再输出 sourcing strategy。”
